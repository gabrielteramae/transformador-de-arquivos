# Transformador de arquivos

API para transformação e conversão de arquivos de dados.

## Sobre

Converta e filtre arquivos CSV, JSON ou XML diretamente pelo browser, sem instalar nada.

## Funcionalidades

- Upload de arquivos CSV, JSON e XML
- Filtro de linhas por expressão
- Seleção de colunas
- Renomeação de colunas
- Conversão entre formatos

## Demo

https://conversor-de-arquivos.up.railway.app

## Stack

- C# / ASP.NET Core
- Docker
- Kubernetes
- Railway

```bash
cd src/DataForge
dotnet run
```
