Como usar as transformações

                
            

            

                

                    
Filtro

                    
Filtra as linhas do arquivo com base no valor de uma coluna.

                    

                        

                            status=ativo
                            linhas onde status é "ativo"
                        

                        

                            idade>18
                            linhas onde idade é maior que 18
                        

                        

                            preco<100
                            linhas onde preço é menor que 100
                        

                    

                

                

                

                    
Colunas

                    
Seleciona apenas as colunas informadas, separadas por vírgula.

                    

                        

                            nome,email,idade
                            retorna só essas três colunas
                        

                    

                

                

                

                    
Renomear colunas

                    
Renomeia colunas usando o formato original:novo, separados por
                        vírgula.

                    

                        

                            name:nome,email:correio
                            renomeia "name" para "nome" e "email" para "correio"
